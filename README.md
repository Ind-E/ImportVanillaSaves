Adds a button in the profile select screen to import save data from unmodded profiles.

Supports game versions v0.103.3 and v0.107.0
<br/>

## Installation

Extract the `.zip` from the [latest release](https://github.com/Ind-E/ImportVanillaSaves/releases/latest) into your Slay the Spire 2 mods folder (create it if it doesn't exist)
- Windows and Linux: `Steam/steamapps/common/Slay the Spire 2/mods/`
- Mac: `Steam/steamapps/common/Slay the Spire 2/SlayTheSpire2.app/Contents/MacOS/mods/`
